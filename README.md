# News Portal

High-performance News Portal built with **Go** (Backend) and **Next.js** (Frontend).

## Architecture

*   **Backend**: Go (Golang) - REST API
*   **Frontend**: Next.js 14+ (TypeScript) - Server-Side Rendering
*   **Database**: PostgreSQL
*   **Infrastructure**: Docker

## Getting Started

### Backend
```bash
cd backend
go run main.go
```

### Frontend
```bash
cd frontend
npm run dev
```
